NFV-Inspector is machine learning ready, VNFs analysis is not a manual task anymore.

NFV-Inspector can automatically run milioons of different test scenarios in an NFV deployment using vendor's exisiting benchmarking tools, and monitor resource utilization and correlate them with VNFs response time. Then, using the aggregated information, it can extract comprhensive knowledge about the NFV system.

It can systematically profile and classify VNFs based on resource capacities, traffic demand rate, underlying system properties, placement of VNFs in the network and many more. It can also extract/calculate the correlation among the QoS metrics and resource utilization of VNFs.

The first public demonstration of this tool will take place in IEEE Conference on Network Function Virtualization and Software Defined Networks in Verona, Italy.
